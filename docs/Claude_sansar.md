# Сансар гар утас худалдаа — Вэб + Админ панелийн техникийн даалгавар

> Энэ баримтыг Claude Code-д шууд хуулж тавьж болно. Бүх UI монгол кириллээр.

## 1. Зорилго

"Сансар гар утас худалдаа" дэлгүүрт зориулсан жижиг, хурдан, мэргэжлийн вэбсайт болон админ панел барих.

- **Олон нийтийн вэб**: зарагдах боломжтой утаснууд, бэлэн үнэ, зээлийн апп бүрээр хуваан төлөх боломжийг харуулна.
- **Админ панел**: эзэн нь нөөцөө (утас нэмэх/засах/зарагдсаныг хасах) удирдана.
- Ижил өгөгдөл нь **тусдаа баригдах Messenger чатботын мэдээллийн эх сурвалж** болж ашиглагдана.

**Гол зарчим:** нэг өгөгдлийн сан → вэб, тооцоолуур, чатбот гурвыг тэжээнэ. Админд нэг л удаа оруулахад бүх газар автоматаар шинэчлэгдэнэ.

## 2. Технологийн стек

- **Next.js** (App Router) + **TypeScript** + **Tailwind CSS**
- **Supabase** — Postgres өгөгдлийн сан, Auth (админ нэвтрэлт), Storage (утасны зураг)
- **Vercel** дээр deploy
- **Mobile-first responsive** — үйлчлүүлэгчид ихэвчлэн Facebook-оос гар утсаар ордог
- Загвар: бараан өнгийн (хар/бараан хүрэн) сэдэв, тод neon-ийн өнгөлбөр (дэлгүүрийн лого/баннертай уялдуулсан), гэхдээ цэвэрхэн, хурдан

## 3. Өгөгдлийн загвар (Supabase)

### `phones` хүснэгт
| багана | төрөл | тайлбар |
|---|---|---|
| id | uuid (pk) | автоматаар |
| brand | text | Apple, Samsung, Xiaomi… |
| model | text | iPhone 15 Pro гэх мэт |
| storage | text | 128GB, 256GB… |
| color | text | өнгө |
| condition | text | 'new' (Шинэ) / 'used' (Хуучин) |
| price | integer | бэлэн үнэ, ₮ |
| image_url | text | Supabase Storage-ийн зураг |
| description | text (nullable) | нэмэлт тайлбар |
| status | text | 'available' / 'sold' (default 'available') |
| created_at | timestamptz | default now() |

### `payment_methods` хүснэгт
| багана | төрөл | тайлбар |
|---|---|---|
| id | uuid (pk) | |
| name | text | Storepay, M Credit, Pocket, LendMN… |
| type | text | 'split' (хүүгүй хуваан төлөлт) / 'loan' (хүүтэй цахим зээл) |
| max_installments | int (nullable) | зөвхөн split-д (ж: Storepay=6, M Credit=4) |
| min_amount | int (nullable) | энэ дүнгээс дээш бол хамаарна (ж: Storepay=100000) |
| note | text (nullable) | монголоор тайлбар |
| logo_url | text (nullable) | аппын лого |
| sort_order | int | харагдах дараалал |
| is_active | bool | default true |

**Анхдагч өгөгдөл (seed) — эзэн засаж болно:**
- Storepay — type=split, max_installments=6, min_amount=100000
- M Credit — type=split, max_installments=4
- Pocket — type=loan
- LendMN — type=loan

### `settings` (key-value эсвэл нэг мөр)
Анхдагч утгууд (дэлгүүрийн баннераас — **зөв эсэхийг эзнээс шалгана уу**):
- shop_name: `Сансар гар утас худалдаа`
- hours: `Өдөр бүр 09:00–21:00`
- phone: `8806-6726`
- address: `Сансарын баянцээлийн туслах замаар өгсөөд Laundry zone угаалгын газраар баруун эргээд, Сансар гар утас худалдааны төв`
- facebook_url: (эзнээс авна)
- logo_url: (зураг upload)

## 4. Хуудаснууд

### Олон нийтийн (public)
1. **Нүүр / каталог** — `status='available'` утаснуудын grid.
   - Карт: зураг, брэнд + загвар, багтаамж, өнгө, шинэ/хуучин badge, бэлэн үнэ, **хүүгүй хуваан төлөлтийн highlight** (ж: "Storepay-ээр сард 315,000₮-аас, хүүгүй").
   - Брэндээр шүүх, нэрээр хайх.
2. **Утасны дэлгэрэнгүй + тооцоолуур** (доорх логик 5-р хэсэгт).
3. **Холбоо барих** — цаг, хаяг, утас, Messenger/Facebook линк (footer-т ч болно).

### Админ (Supabase Auth-аар хамгаалагдсан)
1. **Нэвтрэх** — имэйл + нууц үг.
2. **Утасны жагсаалт** — хүснэгт, status toggle ("Зарагдсан" дарахад status='sold' → вэбээс автоматаар алга).
3. **Утас нэмэх / засах** — бүх талбар + Supabase Storage руу зураг upload.
4. **Төлбөрийн апп удирдах** — апп нэмэх/засах (type, max_installments, min_amount, note, logo).
5. **Тохиргоо** — цаг, утас, хаяг, лого, Facebook линк.

## 5. Тооцооллын логик (хамгийн чухал хэсэг)

**type='split' (хүүгүй апп — Storepay, M Credit):**
- Хэрэв `price >= min_amount` (эсвэл min_amount хоосон бол үргэлж):
  - N = 2 … max_installments хүртэлх сонголтуудыг харуул
  - сарын төлбөр = `Math.ceil(price / N)`
  - **нийт төлбөр = price** (хүү, нэмэгдэл байхгүй)
- Дэлгэрэнгүй хуудсанд slider/сонгогчоор N сонгуулж, "сард [сарын төлбөр]₮ × [N] = [нийт]₮ (хүүгүй)" гэж харуул.
- Каталогийн картад хамгийн их N-ийн сарын төлбөрийг highlight.

**type='loan' (хүүтэй апп — Pocket, LendMN):**
- **Ямар ч сарын дүн тооцоохгүй, зохиохгүй.**
- Зөвхөн харуул: "[Апп нэр]-ээр авах боломжтой. Эцсийн нөхцөл (хүү, хугацаа) таны зээлийн эрхээс хамаарч аппад тогтоно." + лого.
- *Шалтгаан: эдгээр аппын хүү, хугацаа, эрхийг хэрэглэгч бүрд аппад нь тогтоодог тул дэлгүүр урьдчилан мэдэхгүй. Зохиомол дүн харуулбал маргаан үүснэ.*

**Бүх төлбөрийн хэсэгт жижиг тэмдэглэл:** "Үнэ, нөхцөл өөрчлөгдөж болзошгүй. Эцсийн дүнг тухайн аппд баталгаажуулна уу."

## 6. Чатботын холболт (тусдаа эзэн өөрөө барина)

- `GET /api/catalog` — read-only, public, CORS нээлттэй JSON endpoint:
  - `phones`: зөвхөн available — {id, brand, model, storage, color, condition, price, image_url, splits: [{method, N, monthly}], description}
  - `payment_methods`: идэвхтэй жагсаалт
  - `settings`: {shop_name, hours, phone, address}
- Чатботын knowledge base **энэ endpoint-оос уншина** → админ шинэчлэхэд бот автоматаар синк болно.
- Зарагдсан утас энэ endpoint-д орохгүй (бот available-аас санал болгоно).

## 7. UI / форматлал

- Үнэ: мянгатын тусгаарлагч + ₮ (ж: `1,890,000₮`).
- Mobile-first, хурдан ачаалалт.
